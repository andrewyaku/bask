1.1. Интеграция Google Analytics 4:
- Создан файл js/analytics.js с кодом отслеживания GA4
- Measurement ID: G-CXJLTKYPRN
- Подключена аналитика ко всем страницам сайта (~80 файлов)
- Добавлены функции для отслеживания событий:

1.3. Реализация поиска по сайту:
- Создана страница поиска /pages/search.html
- Создан поисковый индекс data/search-index.json (~40 документов)
- Создан js/search.js с гибридной логикой поиска (Lunr.js + JavaScript)
- Добавлена поисковая строка в навигацию на всех страницах (ширина 160px)
- Реализованы функции:
  - Полнотекстовый поиск с поддержкой кириллицы
  - Поиск по частичному совпадению (например, "тренир" → "тренировка", "дрибл" → "дриблинг")
  - Фильтрация по типу контента (правила, статьи, калькуляторы, челленджи, планы)
  - Подсветка найденных слов
  - Сортировка результатов по релевантности
  - Поиск по заголовкам, описаниям, ключевым словам и контенту
  - Отображение результатов с мета-информацией
  - Страница "Ничего не найдено" с подсказками
- Оптимизирован размер поисковой строки для корректного отображения на всех страницах
