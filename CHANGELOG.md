  - Подсветка найденных слов в результатах

- Создан файл `js/analytics.js` для управления аналитикой
- Подключена аналитика ко всем страницам сайта (~80 страниц)
- Добавлены функции отслеживания событий:

  - Оптимизированы цвета для всех разделов сайта (правила, блог, челленджи, планы, статьи)

- Обновлены файлы документации (README.md, SUMMARY.md, CHANGELOG.md)

### Файлы
- `pages/search.html` — страница поиска
