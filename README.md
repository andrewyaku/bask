# BasketGuide - Веб-сайт о баскетболе

BasketGuide - это всесторонний веб-ресурс, посвященный баскетболу. Сайт предоставляет информацию о правилах игры, калькуляторы для расчета параметров, тренировочные планы, челленджи и образовательные материалы для игроков всех уровней.

## Основные разделы сайта

### 1. Калькуляторы
- **Размер мяча** - калькулятор для определения подходящего размера баскетбольного мяча
- **Высота кольца** - расчет правильной высоты баскетбольного кольца
- **Позиция игрока** - определение оптимальной позиции на площадке
- **Индекс эффективности** - расчет статистических показателей игрока

### 2. Правила игры
Подробные объяснения всех основных правил баскетбола:
- Пробежка (Traveling)
- Двойное ведение (Double Dribble)
- Фолы (Fouls)
- Аут (Out of Bounds)
- Зонная защита (Zone Defense)
- Правила пробежки (Backcourt Violation)
- и другие

### 3. Челленджи
Интерактивные задания для улучшения навыков:
- Челленджи ведения мяча
- Челленджи бросков
- Челленджи защиты
- Челленджи правил игры

### 4. Блог
Статьи о различных аспектах баскетбола:
- Техника игры
- Тактика
- Физическая подготовка
- Психология в спорте
- Питание баскетболиста
- История баскетбола

### 5. Тренировочные планы
Программы тренировок для разных целей:
- Развитие прыгучести
- Улучшение броска
- Развитие навыков ведения мяча
- Физическая подготовка

## Архитектура проекта

### Структура папок
```
basketguide/
├── assets/           # Изображения, видео и другие медиафайлы
├── calculators/      # Страницы калькуляторов
├── challenges/       # Страницы челленджей
├── css/              # Стилевые файлы
├── data/             # JSON-файлы с данными
├── js/               # JavaScript-файлы
├── pages/            # Основные HTML-страницы
│   ├── calculator/   # Страницы калькуляторов
│   ├── challenge/    # Страницы челленджей
│   ├── plan/         # Страницы тренировочных планов
│   ├── post/         # Страницы статей блога
│   └── rule/         # Страницы правил
├── plans/            # Страницы тренировочных планов
└── rule/             # Страницы правил (альтернативная папка)
```

### Основные файлы
- `index.html` - главная страница сайта
- `pages/blog.html` - главная страница блога
- `pages/calculators.html` - страница со всеми калькуляторами
- `pages/rules.html` - страница со всеми правилами
- `pages/challenges.html` - страница со всеми челленджами
- `pages/plans.html` - страница со всеми тренировочными планами

## Особенности проекта

### SEO-оптимизация
- Каждая страница имеет уникальные мета-теги
- Поддержка Open Graph и Twitter Card
- Канонические URL
- XML-карта сайта

### Адаптивный дизайн
- Полностью адаптивный интерфейс
- Поддержка мобильных устройств, планшетов и десктопов
- Использование современных CSS-фреймворков

### Интерактивность
- Интерактивные калькуляторы
- Челленджи с проверкой ответов
- Система комментариев
- Функции поделиться статьей

### Аналитика
- Интегрирована Google Analytics 4 (G-CXJLTKYPRN)
- Отслеживание просмотров страниц
- Возможность отслеживания событий (калькуляторы, челленджи, статьи)

## Технологии

## Удаленные компоненты

В ходе развития проекта было принято решение отказаться от видео-контента, и все упоминания видео, включая:
- Страницы с видеоуроками
- Разделы "Видео примеры" в статьях правил
- Связанные видео в статьях блога
- Плейлисты и тьюториалы
- Иконки YouTube

были удалены из проекта для упрощения структуры и улучшения производительности.
