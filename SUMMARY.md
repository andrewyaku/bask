# BasketGuide - Структура проекта

BasketGuide - это всесторонний веб-ресурс, посвященный баскетболу. Сайт предоставляет информацию о правилах игры, калькуляторы для расчета параметров, тренировочные планы, челленджи и образовательные материалы для игроков всех уровней.

## Структура проекта

### Главные страницы
- `index.html` - Главная страница сайта
- `pages/blog.html` - Блог с статьями о баскетболе
- `pages/calculators.html` - Страница со всеми калькуляторами
- `pages/rules.html` - Страница со всеми правилами игры
- `pages/challenges.html` - Страница со всеми челленджами
- `pages/plans.html` - Страница со всеми тренировочными планами
- `pages/about.html` - Страница "О нас"

### Папки проекта

#### `/assets`
- `images/` - Изображения для сайта
- `videos/` - Видеоматериалы (удалены)
- `icons/` - Иконки сайта

#### `/calculators`
- Страницы калькуляторов для расчета параметров игры

#### `/challenges`
- Страницы интерактивных заданий для улучшения навыков

#### `/css`
- `styles.css` - Основные стили сайта
- `responsive.css` - Адаптивные стили
- `loader.css` - Стили для загрузчика

#### `/data`
- `blog.json` - Данные для блога
- `calculators.json` - Данные для калькуляторов
- `challenges.json` - Данные для челленджей
- `plans.json` - Данные для тренировочных планов
- `rules.json` - Данные для правил игры
- `tutorials.json` - Данные для видеоуроков (пустой файл, видео удалены)

#### `/js`
- `main.js` - Основной JavaScript-файл
- `seo.js` - Файл для SEO-функций
- `storage.js` - Файл для хранения данных
- `rules/` - Файлы для работы с правилами
- `calculators/` - Файлы для работы с калькуляторами
- `challenges/` - Файлы для работы с челленджами
- `plans/` - Файлы для работы с тренировочными планами
- `blog/` - Файлы для работы с блогом

#### `/pages`
- `calculator/` - Страницы отдельных калькуляторов
- `challenge/` - Страницы отдельных челленджей
- `plan/` - Страницы отдельных тренировочных планов
- `post/` - Страницы статей блога
- `rule/` - Страницы отдельных правил игры

## Основные разделы сайта

### Калькуляторы
- Размер мяча
- Высота кольца
- Позиция игрока
- Индекс эффективности игрока

### Правила игры
- Пробежка (Traveling)
- Двойное ведение (Double Dribble)
- Фолы (Fouls)
- Аут (Out of Bounds)
- Зонная защита (Zone Defense)
- и другие

### Челленджи
- Челленджи ведения мяча
- Челленджи бросков
- Челленджи защиты
- Челленджи правил игры

### Блог
- Статьи о технике игры
- Статьи о тактике
- Статьи о физической подготовке
- История баскетбола
- Питание баскетболиста
- Психология в спорте

### Тренировочные планы
- Планы для новичков
- Планы для развития прыгучести
- Планы для улучшения броска
- Планы для развития навыков ведения мяча

## Особенности проекта

### Удаленные компоненты
В ходе развития проекта было принято решение отказаться от видео-контента:
- Удалена папка tutorials и все видео-материалы
- Удалены все упоминания видео из HTML-файлов
- Удалены все упоминания видео из JSON-файлов
- Обновлены JavaScript-файлы, убрана функциональность видео
- Удалены секции "Видео примеры" из статей правил
- Удалены иконки YouTube из всех файлов

### Добавленные компоненты
- 8 новых статей в блог
- Обновлены все ссылки на существующие страницы
- Удалены неработающие ссылки
- Улучшена навигация по сайту
- Улучшена видимость элементов интерфейса
